# Contributing

このプロジェクトは小さな静的Webアプリです。

## 改善歓迎ポイント

- 四つ葉検出ロジックの改善
- 誤検出を減らす調整
- UI改善
- スマホブラウザー対応
- README改善
- 実写データを使ったAIモデル化

## 開発方法

```bash
python -m http.server 8080
```

その後、ブラウザーで以下を開きます。

```text
http://localhost:8080/
```

## Pull Requestの方針

- 何を改善したかを簡単に書いてください。
- スマホで確認した場合は機種・ブラウザーも書いてください。
- 検出ロジックを変更した場合は、誤検出が増えていないか確認してください。
