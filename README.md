# 四つ葉ファインダー

スマホのカメラでクローバーを映し、四つ葉らしい候補を赤・黄色の丸で表示する軽量Webアプリです。

## v0.2.0の主な変更

スマホでの実使用を優先してUIを整理しました。

- 「端末内解析」表示を削除
- 「今すぐ解析」ボタンを削除
- スコア・候補数・解析時間表示を削除
- 判定説明パネルを削除
- 緑抽出マスク表示を削除
- カメラ開始・停止ボタンを押した感触のあるデザインに変更
- 最初の画面でカメラ画面と主要操作ボタンが見える構成に変更
- スライダー調整を3段階ボタンに変更
- 縦持ち・横持ちの両方に対応

## 使い方

1. スマホでアプリURLを開きます。
2. 「カメラ開始」を押します。
3. クローバーの群生にカメラを向けます。
4. 赤い丸または黄色い丸が出た場所を目視確認します。
5. 候補が多すぎる場合は「厳しめ」を選びます。
6. 候補が出ない場合は「ゆるめ」を選びます。

## GitHub Pagesで公開する

`Settings` → `Pages` で以下を設定します。

```text
Source: Deploy from a branch
Branch: main
Folder: /(root)
```

## 注意

このアプリは四つ葉を確実に判定するAIではなく、候補表示型の画像処理プロトタイプです。
画像は外部サーバーへ送信せず、端末内の JavaScript と Canvas だけで処理します。
